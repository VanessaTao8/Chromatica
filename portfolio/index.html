Large Language Model for Generating Twitter Data in Sentiment Analysis

  • Finished
  • Sep-2023
  • Dec-2023

We explore the use of LLMs in generating Twitter data for sentiment analysis. Notably, the state-of-the-art method, Attrprompt, struggles to generate quality Twitter data. To address this, we introduce the Fact-Imagination Prompt, which prompts LLMs to imagine potential reasons and events underlying tweets. These imagined scenarios are then used to produce new tweets.

Modeling the Volatility Index using Time Series and NN methods

  • Finished
  • Sep-2023
  • Dec-2023

The project tries to model the VIX time series, starting by fitting the basic conditional mean models and jointly modeling the conditional mean and conditional variance of the VIX time series using the joint ARIMA+GARCH models. A more advanced transformer-based Neural Network model called Temporal Fusion Transformer is latered deployed to incorporate volatility of different industry classes in predicting VIX.

Distinguishing AI-Generated Text from Human-Authored Text

  • Finished
  • Feb-2024
  • May-2024

This project explores the use of both supervised and unsupervised models to distinguish between AI-generated and human-authored text. With the rise of large language models (LLMs), particularly tools like ChatGPT, distinguishing machine-generated content has become increasingly important. The dataset used for this study contained 788,922 unique values with information on text type (AI or human), prompt ID, and word count.

Prediction Analysis & Visualization on CDC’s Chronic Data

  • Finished
  • Feb-2024
  • May-2024

This project focuses on developing an interactive map to visualize the spread of chronic diseases across the U.S., aiding public health efforts in resource allocation, planning, and prevention. Using CDC data from 2007 to 2021, which includes over 1.2 million records, the project employs Exponential Smoothing, Grey Prediction, and ARIMA models to forecast state-level disease trends.
